# ☕ Black Coffee

A landing page for a coffee shop, featuring custom typography (Dana font) and a warm, dark aesthetic.

## Features
- Custom web font integration (Dana)
- Responsive coffee shop landing page
- Image gallery / menu section

## Tech Stack
HTML5 · CSS3 · JavaScript (Vanilla)

## Project Structure
black-coffee/
- index.html   -> Main page
- app.js       -> Interactivity
- style.css    -> Styling
- images/      -> Site images
- fonts/Dana/  -> Custom font files

## How to Run
git clone https://github.com/hj904549-wq/black-coffee.git

Open index.html in your browser.
